## Submission-Ready Definition

A submission is finals-ready when all are true:

1. `npm run judge:rehearse` reports zero failed runs.
2. `npm run judge:prove` prints matching summary/verifier hashes.
3. Baseline and ROI tables are filled with measured values.
4. At least one external validation artifact is attached.
5. Demo path and fallback path are both rehearsed.
